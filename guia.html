Como Colocar seu Site no Ar - Guia Prático

Hospedagem gratuita para seus 54 arquivos (10 matérias × 5 arquivos + 4 arquivos extras)

Primeiro: Organize seus arquivos

Antes de tudo, organize todos os seus arquivos em pastas. Faça assim:

  • Pasta principal: Coloque todos os arquivos do seu site aqui
  • Subpastas: Crie uma pasta para cada matéria (10 pastas)
  • Arquivos principais: index.html e outros 3 arquivos na raiz

Dica importante:

O arquivo principal DEVE se chamar index.html - é esse que o navegador vai abrir automaticamente quando alguém acessar seu site.

Escolha uma plataforma gratuita

Você tem várias opções para hospedar seu site de graça. Escolha uma:

Opção 1: GitHub Pages (Recomendado)

  1. Crie uma conta no GitHub (é grátis)
  2. Crie um novo repositório com o nome: seuusuario.github.io (substitua "seuusuario" pelo seu nome de usuário)
  3. Faça upload de todos os seus arquivos para esse repositório
  4. Pronto! Seu site estará em: https://seuusuario.github.io

Opção 2: Netlify

  1. Vá para Netlify.com
  2. Faça login com sua conta do GitHub, Google ou crie uma conta
  3. Arraste e solte a pasta com todos os seus arquivos na área indicada
  4. O Netlify vai gerar um link único para seu site automaticamente

Opção 3: Vercel

  1. Acesse Vercel.com
  2. Faça login com sua conta do GitHub
  3. Clique em "Import Project" e selecione a pasta com seus arquivos
  4. Em segundos, seu site estará no ar com um link único

Passo a Passo Detalhado (GitHub Pages)

Passo 1: Crie sua conta no GitHub

Vá para github.com e crie uma conta. É rápido e gratuito.

Passo 2: Crie um novo repositório

Depois de fazer login, clique no ícone "+" no canto superior direito e selecione "New repository".

Dê o nome EXATO: seunomedeusuario.github.io

Marque a opção "Public" e clique em "Create repository".

Passo 3: Faça upload dos arquivos

Na página do repositório que você criou, clique em "Add file" → "Upload files".

Arraste TODOS os seus arquivos e pastas para a área indicada ou clique em "choose your files".

Não esqueça de incluir o arquivo index.html na raiz!

Passo 4: Commit das alterações

Role para baixo, escreva uma mensagem no campo "Commit changes" (pode ser "Primeiro upload")

Clique no botão "Commit changes".

Passo 5: Acesse seu site

Aguarde alguns minutos e depois acesse: https://seunomedeusuario.github.io

Pronto! Seu site está no ar para todo mundo ver!

Dicas Finais Importantes

Verifique os links internos

Teste todos os links entre suas páginas para garantir que estão funcionando corretamente.

Organização é fundamental

Mantenha uma estrutura de pastas organizada para não se perder com tantos arquivos.

Nomeie os arquivos corretamente

Use apenas letras, números e hífens nos nomes dos arquivos. Evite espaços e caracteres especiais.

Quer um domínio personalizado?

Mais tarde, você pode comprar um domínio (como meusite.com) e configurar no GitHub Pages. Mas por enquanto, o link gratuito já resolve!
