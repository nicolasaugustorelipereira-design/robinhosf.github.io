Circuito em Série

Os componentes são conectados um após o outro, formando um único caminho para a corrente elétrica:cite[4]:cite[7].

Características Principais:cite[7]:cite[8]:

  • Corrente (A): A mesma corrente flui por todos os componentes. Itotal = I1 = I2 = I3
  • Tensão (V): A tensão total é a soma das tensões em cada componente. Vtotal = V1 + V2 + V3
  • Resistência Total (Ω): É a soma de todas as resistências. Rtotal = R1 + R2 + R3 + ...
  • Ponto Fraco: Se uma lâmpada queimar ou um componente for desconectado, o circuito é interrompido e todas as lâmpadas se apagam:cite[4].

Circuitos em Série e em Paralelo

A forma como os componentes (como lâmpadas e resistores) são conectados em um circuito define seu comportamento. As duas configurações mais básicas são série e paralelo.

Circuito em Série

Os componentes são conectados um após o outro, formando um único caminho para a corrente elétrica.

Características Principais:

  • Corrente (A): A mesma corrente flui por todos os componentes. Itotal = I1 = I2 = I3
  • Tensão (V): A tensão total é a soma das tensões em cada componente. Vtotal = V1 + V2 + V3
  • Resistência Total (Ω): É a soma de todas as resistências. Rtotal = R1 + R2 + R3 + ...
  • Ponto Fraco: Se uma lâmpada queimar ou um componente for desconectado, o circuito é interrompido e todas as lâmpadas se apagam.

Circuito em Paralelo

Os componentes são conectados lado a lado, oferecendo múltiplos caminhos independentes para a corrente.

Características Principais:

  • Tensão (V): A tensão é a mesma para todos os componentes. Vtotal = V1 = V2 = V3
  • Corrente (A): A corrente total é a soma das correntes em cada ramo. Itotal = I1 + I2 + I3 + ...
  • Resistência Total (Ω): A resistência total é sempre menor que a menor resistência individual. 1/Rtotal = 1/R1 + 1/R2 + 1/R3 + ...
  • Vantagem Chave: Se uma lâmpada queimar, os outros caminhos continuam funcionando normalmente. É por isso que as instalações elétricas das nossas casas são feitas em paralelo.
