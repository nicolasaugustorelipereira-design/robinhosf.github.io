2º Bimestre

Após criar as tabelas e definir suas colunas e tipos de dados, o próximo passo em um banco de dados é inserir registros, ou seja, adicionar as informações que irão compor o sistema. No MySQL, essa operação é feita com o comando INSERT INTO.

O comando INSERT INTO pode ser usado para inserir dados linha por linha ou em lotes. Cada linha inserida passa a fazer parte da tabela de forma permanente, desde que não ocorram erros.

A estrutura básica para inserir um único registro é: 

                INSERT INTO NomeDaTabela (coluna1, coluna2, coluna3)
                VALUES (valor1, valor2, valor3);

Exemplo prático: inserindo um aluno na tabela Aluno. 

                INSERT INTO Aluno (matricula, nome, idade, curso)
                VALUES (1001, 'Ana Silva', 20, 'Informática');

Nesse exemplo, o aluno Ana Silva foi adicionado com matrícula 1001, idade 20 e curso Informática.

Inserindo vários registros ao mesmo tempo

Para otimizar a inserção, é possível adicionar múltiplos registros de uma só vez, separando os valores com vírgulas: 

                INSERT INTO Aluno (matricula, nome, idade, curso)
                VALUES 
                    (1002, 'Bruno Souza', 21, 'Redes'),
                    (1003, 'Carla Dias', 19, 'Desenvolvimento'),
                    (1004, 'Diego Lima', 22, 'Informática');

Esse formato é muito mais eficiente quando se tem uma grande quantidade de dados para registrar.

Cuidados importantes ao inserir dados

  • A ordem das colunas deve corresponder à ordem dos valores informados.
  • Os valores devem respeitar os tipos de dados definidos na criação da tabela.
  • Se a tabela possuir uma chave primária, o valor dessa coluna deve ser único.

Exemplo de erro por duplicidade: 

                -- Isso causará erro se a matrícula 1001 já existir
                INSERT INTO Aluno (matricula, nome, idade, curso)
                VALUES (1001, 'Eduardo Alves', 23, 'Segurança da Informação');

Usando o AUTO_INCREMENT

Para evitar erros de duplicidade e facilitar a inserção de novos dados, o MySQL oferece o recurso AUTO_INCREMENT. Ele gera automaticamente valores únicos e sequenciais para uma coluna, geralmente utilizada como chave primária.

Exemplo de criação de tabela com AUTO_INCREMENT: 

                CREATE TABLE Aluno (
                    matricula INT PRIMARY KEY AUTO_INCREMENT,
                    nome VARCHAR(100),
                    idade INT,
                    curso VARCHAR(50)
                );

Com essa configuração, não é necessário informar o valor da matrícula ao inserir um novo aluno: 

                INSERT INTO Aluno (nome, idade, curso)
                VALUES ('Fernanda Rocha', 21, 'Análise de Sistemas');

O sistema automaticamente atribuirá o próximo número disponível. Por exemplo, se o último aluno inserido tinha matrícula 1004, o próximo será 1005.

Dessa forma, o AUTO_INCREMENT evita conflitos, garante unicidade e simplifica a manutenção dos registros no banco de dados.
