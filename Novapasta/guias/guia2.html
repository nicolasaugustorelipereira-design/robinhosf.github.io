Introdução ao PHP e Bancos de Dados

PHP é uma linguagem de script do lado do servidor amplamente utilizada para desenvolvimento web. Quando combinado com um sistema de gerenciamento de banco de dados (como MySQL), o PHP permite criar aplicações web dinâmicas e interativas.

Importante: Para usar PHP com banco de dados online, você precisará de uma hospedagem que suporte PHP e MySQL.

Por que usar PHP com banco de dados?

  • Armazenamento persistente de dados
  • Recuperação e manipulação eficiente de informações
  • Criação de aplicações web dinâmicas
  • Gerenciamento de usuários e sessões
  • Análise e relatórios de dados

Configuração do Ambiente

Antes de começar, você precisa configurar seu ambiente de desenvolvimento:

Ambiente Local (Desenvolvimento)

  • XAMPP - Pacote que inclui Apache, PHP e MySQL
  • WAMP - Alternativa para Windows
  • MAMP - Para usuários macOS
  • LAMP - Para Linux

Ambiente Online (Produção)

Para hospedar seu site online, você precisará de:

  • Servidor de hospedagem com suporte a PHP
  • Banco de dados MySQL (ou similar)
  • Acesso FTP ou painel de controle (como cPanel)

Criando um Banco de Dados

Existem duas formas principais de criar um banco de dados para usar com PHP:

1. Usando PHPMyAdmin (Interface Gráfica)

O PHPMyAdmin é uma ferramenta web popular para gerenciar bancos de dados MySQL.

  1. Acesse o PHPMyAdmin através do seu painel de hospedagem
  2. Clique em "Novo" para criar um novo banco de dados
  3. Digite um nome para o banco de dados e clique em "Criar"
  4. Crie tabelas e defina a estrutura dos dados

2. Usando PHP (Programaticamente)

Você também pode criar um banco de dados diretamente através de código PHP:

// Conectar ao servidor MySQL (sem especificar um banco de dados)
$servername = "localhost";
$username = "seu_usuario";
$password = "sua_senha";

// Criar conexão
$conn = new mysqli($servername, $username, $password);

// Verificar conexão
if ($conn->connect_error) {
  die("Conexão falhou: " . $conn->connect_error);
}

// Criar banco de dados
$sql = "CREATE DATABASE meu_banco_de_dados";
if ($conn->query($sql) === TRUE) {
  echo "Banco de dados criado com sucesso";
} else {
  echo "Erro ao criar banco de dados: " . $conn->error;
}

$conn->close();

Conectando PHP ao Banco de Dados

Após criar o banco de dados, você precisa conectar seu código PHP a ele:

// Detalhes de conexão
$servername = "localhost";
$username = "seu_usuario";
$password = "sua_senha";
$dbname = "meu_banco_de_dados";

// Criar conexão
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexão
if ($conn->connect_error) {
  die("Conexão falhou: " . $conn->connect_error);
}
echo "Conectado com sucesso";
Atenção: Nunca armazene informações de conexão diretamente no código em produção. Use variáveis de ambiente ou arquivos de configuração protegidos.

Hospedando Online

Para hospedar seu site com banco de dados PHP online:

1. Escolha uma Hospedagem

Selecione um provedor de hospedagem que ofereça:

  • Suporte a PHP (versão compatível com seu código)
  • Banco de dados MySQL
  • Painel de controle (cPanel, Plesk, etc.)
  • Acesso FTP/SFTP

2. Upload dos Arquivos

Use um cliente FTP (como FileZilla) ou o gerenciador de arquivos do painel de controle para fazer upload:

  1. Conecte-se ao servidor via FTP usando as credenciais fornecidas
  2. Navegue até a pasta pública (geralmente public_html, www ou htdocs)
  3. Faça upload de todos os seus arquivos PHP, HTML, CSS e JavaScript

3. Configurar o Banco de Dados Online

  1. Acesse o painel de controle da hospedagem
  2. Crie um banco de dados MySQL
  3. Anote o nome do banco, usuário, senha e servidor (host)
  4. Importe sua estrutura de banco de dados (se necessário)
  5. Atualize as informações de conexão no seu código PHP
Dica: Muitos provedores de hospedagem usam "localhost" como servidor de banco de dados, mas alguns podem usar um endereço específico. Verifique a documentação do seu provedor.

Exemplo Prático Completo

Aqui está um exemplo completo de um sistema simples de cadastro de usuários:

Estrutura do Banco de Dados

-- Criar tabela de usuários
CREATE TABLE usuarios (
  id INT(6) UNSIGNED AUTO_INCREMENT PRIMARY KEY,
  nome VARCHAR(50) NOT NULL,
  email VARCHAR(50) NOT NULL,
  data_cadastro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Código PHP para Inserir Dados

<?php
// Configurações do banco de dados
$servername = "localhost";
$username = "seu_usuario";
$password = "sua_senha";
$dbname = "meu_banco_de_dados";

// Criar conexão
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexão
if ($conn->connect_error) {
  die("Conexão falhou: " . $conn->connect_error);
}

// Processar formulário
if ($_SERVER["REQUEST_METHOD"] == "POST") {
  $nome = $_POST["nome"];
  $email = $_POST["email"];

  // Inserir dados no banco
  $sql = "INSERT INTO usuarios (nome, email) VALUES ('$nome', '$email')";

  if ($conn->query($sql) === TRUE) {
    echo "Novo registro criado com sucesso";
  } else {
    echo "Erro: " . $sql . "<br>" . $conn->error;
  }
}

$conn->close();
?>
Importante: Este exemplo usa consultas diretas para simplificar, mas em produção você deve usar prepared statements para prevenir ataques de SQL injection.
